QUESTIONS

Q1
Create the form structure

Create an HTML file named index.html with the correct HTML5 boilerplate. Inside the <body>, add a <form> element with action="#" and method="post". Add an <h1> heading that reads "Student Registration Form".

Q2
Text inputs with labels

Add two type="text" inputs for First Name and Last Name. Each must have a <label> correctly linked using for and id, and a placeholder attribute.

Q3
Number input

Add an Age field using type="number". Set min="10" and max="99". Test in your browser — what happens if you type a letter instead of a number?

Q4
Email input

Add an Email Address field using type="email". Try submitting the form without an @ symbol — what does the browser do automatically?

Q5
Telephone input

Add a Phone Number field using type="tel". Add a placeholder showing the expected format, e.g. +234 800 000 0000.

Q6
Password input

Add a Password field using type="password" with minlength="8". Observe how the browser hides the characters as you type.

Q7
Radio buttons — gender

Add a Gender group using type="radio" with the options: Male, Female, Prefer not to say. All three must share the same name attribute. Why is this important?

Q8
Checkboxes — courses

Add a Courses Interested In group using type="checkbox" with at least 4 options of your choice. Demonstrate that more than one can be selected at the same time.

Q9
Submit and reset buttons

Add two buttons at the bottom of the form: one type="submit" labelled "Register Now" and one type="reset" labelled "Clear Form". Test both buttons in your browser.

Q10 BONUS
Required fields + reflection

Add the required attribute to at least 4 fields. Then, in an HTML comment inside your file (<!-- -->), write one sentence explaining the difference between a radio button and a checkbox.
